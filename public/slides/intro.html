HTMX? 0
  • Creator - Carson Gross. Version 1 released November 2020.
  • A 15kb minified, dependency free JS library to develop modern apps. "A library that completes HTML" - CG 2023
  • Access modern browser features directly from HTML, instead of using JS.
  • Gives you access to AJAX, CSS Transitions, WebSockets, Server Sent Events and more using HTML attributes
  • HTMX can be used alone, but can also be integrated into React and other frameworks/libraries
  • Successor to intercooler.js, and gaining a lot of traction
  • Typically respond with HTML instead of JSON
