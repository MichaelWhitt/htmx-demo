Motivation for HTMX 1
  • HTML only offers two primary hypermedia controls that can issue HTTP requests: anchors and forms...why not every element?
  • Why should only click and submit trigger them?
  • Why should only GET be availabe for < a > and POST for < form >?
  • Why can you only replace the entire screen?
    • < a > href="https://hypermedia.systems/" < / a >
    • -Browser issues an HTTP request to the URL
    • -Browser will load the HTML body of the HTTP response into the browser window, replacing the current document
    Removing these arbitrary constraints to make HTML itself more interactive is the core of what HTMX does.
